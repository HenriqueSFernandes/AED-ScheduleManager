◆ DisplayStudentSchedule()

void ControlUnit::DisplayStudentSchedule ( )

Displays the schedule of a student.

Getting the schedule is O(log n + m * k ) where n is the total number of students, m is the number of student groups each student has k is the lessons in a student group. Basically O(log n) as the other input is almost constant for each student and very small compared to n. Aditionally after getting the schedule it has to be displayed and that takes time which is not taken into consideration given its small impact on the overall performance.
